使い方

まずはいつも通り

% git clone https://github.com/uryu1994/Uryutter.git

Eclipseの場合

  1. Import-Maven/Existing Maven Projectsを選択
  2. cloneしたリポジトリのルートディレクトリを選択してFinishを選択
  3. あとはmaven installで自動的に依存性を解決してくれます

NetBeansの場合

普通にプロジェクトを開くで開けばMavenプロジェクトとして認識してくれます
